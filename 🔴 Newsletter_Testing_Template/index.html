MICROSOFT と RED HAT がハイブリッドクラウド分野での提供を発表！

共同発表にて、Microsoft は Microsoft Azure 上のエンタープライズ Linux ワークロードの優先的選択肢として Red Hat Enterprise Linuxを提供することを発表。さらに両社は、プライベートクラウドとパブリッククラウドに渡る Red Hat ソフトウェア上でのアプリケーションの構築、配備、および管理に関しても、共同作業を行うことを発表しました。協業の全貌についてはプレスリリースをご覧ください。

主な提携エリアについて、レッドハット販売パートナー様向けに補足情報を加えて、ご紹介します。

  • Microsoft AzureユーザーがRed Hatソリューションをネイティブに利用可能に。今後数週間以内にMicrosoft AzureはRed Hat Certified Cloud and Service Providerとして認定され、お客様はRed Hat Enterprise LinuxアプリケーションやワークロードをMicrosoft Azure上で実行できるようになります。
  • Microsoftとの.NETに関する協業により、Red Hat Enterprise Linux (RHEL)、Red Hat Enterprise Linux Atomic Host、そして OpenShift by Red Hat の各 Red Hat 製品で開発者が.NETテクノロジーを利用できるようになります。
  • ハイブリッド環境全体に渡るエンタープライズグレードの統合サポートの提供。具体的には、Red Hat 製品がMicrosoft Azure上で提供およびサポートされ、Microsoft Windows は Red Hat Enterprise Linux OpenStack Platform および Red Hat Enterprise Linuxのサポートを提供します。
  • ハイブリッドクラウドデプロイメント全体で統一されたワークロード管理を実現。具体的には、Red Hat CloudFormsがMicrosoft AzureおよびMicrosoft System Center Virtual Machine Managerと共に動作し、Red Hat CloudFormsのユーザーはHyper-VとMicrosoft Azureの両方でRed Hat Enterprise Linuxを管理できるようになります。
